# 🚀 Review Dashboard UI

A modern dashboard UI for analyzing customer reviews across platforms.
This project focuses on **frontend design and user experience** for review analysis.

---

## 📌 Overview

This dashboard helps visualize customer feedback using charts and structured UI components.
It is designed to show how review data can be organized into meaningful insights.

---

## ✨ Features

* 📊 **Sentiment Analysis**

  * Displays Positive, Negative, and Neutral reviews using charts

* ⭐ **Top Issues**

  * Shows common issues like Delivery, Quality, and Size
  * Includes visual indicators (progress bars)

* 📈 **Platform Comparison**

  * Bar chart comparing reviews across platforms
  * Example: Amazon, Flipkart, Website

* 📥 **Unified Review Inbox**

  * Displays recent reviews in a single view
  * Includes ratings and sentiment tags

* 💬 **Response Tool**

  * Simple UI to reply to customer reviews

* 🤖 **AI Insights Section (UI)**

  * Static insights displayed at the top of the dashboard

* 🌙 **Dark Mode**

  * Toggle between light and dark themes
  * Theme preference saved using localStorage
